一、vue创建项目
1.查看版本
	vue --version
2.卸载全局vue
	npm uninstall -g vue-cli
3.安装vue3
	npm install -g @vue/cli
	---vue3同时使用vue2的需要安装桥接
	npm install -g @vue/cli-init

vue2创建项目：	vue init webpack my_project
vue3创建项目：	vue create my_project

vue界面管理工具：	vue ui
	

二、添加jar包：
mvn install:install-file -DgroupId=com.oracle -DartifactId=ojdbc14 -Dversion=11.2.0.1.0 -Dpackaging=jar -Dfile=C:\\Users\\86188\\Desktop\\ojdbc14-11.2.0.1.0.jar
